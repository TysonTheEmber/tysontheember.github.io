# EmberForge Documentation

Welcome to the official documentation hub for **EmberForge** - innovative Minecraft mods and creative projects by TysonTheEmber.

## Featured Projects

Use the top navigation bar to access documentation for each project:

### Embers Text API
Powerful library for creating animated text overlays, cinematic messaging, and dynamic in-game displays with beautiful styling effects.

**Key Features:**
- Dynamic typewriter and shake animations
- Multi-stop gradients and custom fonts
- Flexible positioning with 9 anchor points
- NBT command support for scripting

[View Documentation](/embers-text-api/intro)

### Aperture API
Professional cinematic camera tools with an in-game editor, advanced path system, and powerful interpolation capabilities.

**Key Features:**
- Multiple interpolation methods (Catmull-Rom, Bezier, cosine, step)
- Interactive in-game editor with visual gizmos
- Constant-speed path preview
- Path export functionality

[View Documentation](/aperture-api/intro)
