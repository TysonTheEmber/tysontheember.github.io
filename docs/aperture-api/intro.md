# Aperture API

**Aperture API** is a comprehensive Forge cinematic camera tooling mod with in-game editing capabilities and camera path export functionality.

## Overview

Create professional-quality Minecraft cinematics with intuitive camera path controls, smooth animations, and powerful editing tools.

## Features

- In-game camera path editor
- Smooth camera movements
- Path export functionality
- Cinematic camera controls
- Timeline-based editing
- Keyframe animation system
