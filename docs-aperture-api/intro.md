# Aperture API

**Aperture API** is a Minecraft Forge 1.20.1 mod that provides professional cinematic camera tools with an in-game editor, advanced path system, and powerful interpolation capabilities.

## Overview

Create stunning Minecraft cinematics with a fully-featured camera path system supporting multiple interpolation methods, real-time preview, and an intuitive in-game editor.

## Key Features

- **Advanced Path System**: Support for Catmull-Rom, Bezier, cosine, and step interpolation
- **In-Game Editor**: Interactive editor screens with visual gizmos and overlay rendering
- **Playback Tools**: Constant-speed path preview and animation controls
- **Command Interface**: Essential `/aperture` and `/camera` commands for path management
- **Developer API**: Clean, minimal public API for mod integration
- **Path Export**: Export camera paths for external use

## Technical Details

- **Minecraft Version**: 1.20.1 (Forge)
- **Java Version**: 17
- **License**: GPL-3.0

## Installation

1. Download the latest version from:
   - [Modrinth](https://modrinth.com/mod/aperture-api)
   - [CurseForge](https://www.curseforge.com/minecraft/mc-mods/aperture-api)
   - [GitHub](https://github.com/TysonTheEmber/Aperture-API)

2. Place the `.jar` file in your Minecraft `mods` folder

3. Launch Minecraft with Forge 1.20.1

## Quick Start

1. Install the mod in your Forge 1.20.1 instance
2. Launch Minecraft and load a world
3. Use `/aperture` to access the camera editor
4. Use `/camera` commands to manage playback and paths
5. Create camera paths using the in-game editor
