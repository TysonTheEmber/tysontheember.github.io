# Commands

Loot Log provides a single client-side command for managing overrides.

---

## /lootlog reload

Reloads all JSON override files from `config/lootlog/overrides/` without restarting the game.

**Usage:** `/lootlog reload`

**Permission:** None (client-side command)

**Feedback:** "Loot Log overrides reloaded."

:::tip
Use this command after editing override files to apply changes instantly. Config settings (position, animation, etc.) are managed separately through the config file or YACL GUI.
:::
