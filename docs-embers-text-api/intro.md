# Embers Text API

**Embers Text API** is a powerful Minecraft mod that provides animated in-game overlays, banners, and cinematic messaging capabilities.

## Overview

Create stunning visual text effects in your Minecraft world with dynamic animations, custom fonts, and flexible positioning options.

## Features

- Animated in-game overlays
- Customizable banners
- Cinematic text messaging
- Custom font support
- NBT and JSON configuration
- Command-based control
