# Spelunkery+

**Spelunkery+** is a compatibility expansion mod that enhances mining progression and underground exploration with new loot and features.

## Overview

Expand your underground adventures with enhanced mining mechanics, new loot tables, and improved compatibility with other mining-focused mods.

## Features

- Enhanced mining progression
- Expanded underground loot tables
- Compatibility with popular mining mods
- New underground content
- Balanced progression system

## Installation
