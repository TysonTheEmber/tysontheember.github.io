Frequently Asked Questions

These are some of the most asked questions relating my modpacks, if you need more information please contact support.

1. There are so many mods, how can anyone run this?

Although our modpacks are on the heavier side, most of the time people can run them fairly easily. This is because a big part of the modpack is optimization, we spend a lot of time making sure that it runs smoothly... even on low end PCs. Remember to allocate enough RAM though. (12-16gb RAM recommended)

2. Why isn't there a server for (insert pack name)?

Our servers may change over time, some may switch packs. If you are unable to join a server with the correct pack, please update or contact support.

3. I found a bug, what do I do?

If you ever find a bug or exploit please contact me and explain what happened and how you did it. Please check if the bug can be replicated before reporting it. If so, please report it via github and/or discord.

4. Can I invite my friends?

As long as everyone follows the rules and uses common sense they can join. It would be amazing if you can help us grow and expand. Please make sure that everyone joins the Discord though.

5. How can I help?

A big reason I am able to keep these servers up is because of the support of Ko-Fi. Running these server takes alot of time and effort. The best way of supporting me is by becoming a Ko-Fi supporter. Doing this will also grant you a rank on ALL our servers and give you access to special perks and in game(notjust server) rewards.
